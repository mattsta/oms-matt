Client installation:

1) Make so that the bin-debug directory be accessible from http
2) Install OneTeam modified version into firefox by clicking this link: http://dev3.process-one.net/~prefiks/flash-jingle/oneteam.xpi
3) Open page test.html into your web browser via HTTP
4) See the Flash applet to set up the rtmp server and application address
4) connect OneTeam to your xmpp server


Server installation:

1) check it out from : https://svn.process-one.net/oms/branches/oms.new/ if you are reading this, this must already be done
2) Build the server: ./configure; make all
3) Run the server: ./omsctl start
